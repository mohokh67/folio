# Folio

Personal expense tracker built with Flutter. Supports recurring expenses, category management, a calendar view, and a summary dashboard.

## Features

- **Calendar view** — browse expenses by day/month
- **Recurring expenses** — create repeating series with edit/cancel-from-date support
- **Categories** — custom categories with colors and icons
- **Summary tab** — monthly progress, category breakdown, upcoming expenses

## Getting Started

```bash
flutter pub get
flutter run
```

Requires Flutter 3.x. Tested on iOS and Android.
